The Basic Inform extension, included in all projects. This is version 10.2.0.

  • Preamble - Basic Inform is like a boot program for a computer that is starting up: it sets up the compiler to implement the familiar language, beginning with basic verbs and use options.

  • Miscellaneous Definitions - A miscellany of properties, variables, rulebooks and activities. Whereas the Standard Rules for interactive fiction are luxuriant in providing these, the Basic Inform kernel is as small as it sensibly can be, with just a few of each.

  • Phrase Definitions - The phrases making up the basic Inform language, and in terms of which all other phrases and rules are defined.

  • Glulx and Glk - Support for Glulx and Glk interpreter systems.

  • Adjectival Definitions - All the built-in adjectives on Basic Inform, from even and odd (for numbers) to symmetric (for relations).

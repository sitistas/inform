§1. Infglk. This segment provides the infglk.h header which defines the Glk constants and API functions.

This is an almost verbatim copy of an invaluable I6 header file originally put together by John Cater but now maintained by Andrew Plotkin. The routines are convenient to have on hand, and also provide a canonical set of I6 names for the many gestalt and other codes.

Some things have been commented out:

- the System_file directive (no longer supported by Inform 10) - various glk functions which are redefined in Glk.i6t

The current version can be obtained at https://github.com/erkyrath/glk-dev/blob/master/dispatch_dump/infglk.h
